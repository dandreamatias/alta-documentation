Alpha!

Alta-UI it's still an alpha.
You can install and play with Alta-UI but be aware that it's not ready for production.
There is untested code and the breaking changes are frequent. If you are only interested in the progress you can use this website to see the components and how implement them.

Install

npm i alta-ui

Import in project

import 'alta-ui/dist/alta-ui/alta-ui.esm.js';
